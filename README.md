#Olist Ecommerce Data Analysis

##Project Approach

This project follows a stakeholder-first analytical approach, where business questions are defined before analysis begins.

###Step 1: Stakeholder Simulation using LLM

Instead of manually defining the problem, a Large Language Model was used to simulate a Head of Growth stakeholder.The dataset context was provided, and the model was prompted to:

-Define business challenges
-Frame a problem statement
-Generate key analytical questions

This helped replicate a real-world scenario where analysts receive ambiguous or high-level business requirements.
